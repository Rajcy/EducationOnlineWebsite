代码注释

Swift编译器从源代码中生成可执行代码。 为此,它运用了详细的规则集。 有时,这些规则集会模糊了 为什么你要以这种特定的方式写你的代码,甚至你要解决的是什么问题。 为了避免这种情况,最好能够记录的写的代码。下一个人阅读时能够理解你的代码。 毕竟,未来你也可能遇到这种情况。 :]

Swift,像大多数其他编程语言,允许您通过使用注解记录代码。 这让你直接在代码注解文字,而这经常被编译器忽略。

第一种方法写注解是这样的:

// 这是一个注解。 不可执行。

这是一个 单行注释. 你可以把它们累计写成段落:

/ /这也是一条注解。
/ /到多个行。

然而,有一个更好的方法来写注释,可跨越多个行。 像这样:

/ *这也是一条注解。
在许多. .
许多……
多行。 * /

这是一个 多行注释. 开始用/* 结束用*/. 简单吧!

在有需要记录你的代码是使用代码注释,解释你的推理,否则你同事只会对它一笑置之。 :]

算术运算

当你把一个或多个数据,把它们合成一个数据,这就叫做运算.

理解运算最简单的方法是利用算术。 添加运算需要两个数字并将它们转换为两个数之和。 减法运算需要两个数字并将它们转换为两个数字之差。

你会发现简单的算术都用在你的应用程序上，计算出你‘偏爱’的数字“后,计算按钮或一个窗口的正确的大小和位置,数字确实是无处不在!

在本节中, 介于对算术的运用，您将了解Swift提供的各种算术运算。 在后续部分中,可以看到多种算术和运算。

简单的运算

Swift中所有运算使用一个称为运算符的符号来表示他们执行的运算类型。

考虑你在早期学校学习的四个算术运算:加法,减法,乘法和除法。 对于这些简单的操作,Swift使用如下的操作符:

  • 添加: +
  • 减: -
  • 乘: *
  • 除: /

使用这些操作符一样:

文件中的每一行被称为是一个 表达式.每一个表达式都有一个值。 在这些情况下,所有四个表达式有相同的值:8。 你写的代码来执行这些算术运算就像你使用纸和笔运算一样。

小数

以上的所有操作使用整数,正式名称为 整数.然而,你将会知道,并不是每一个数字都是整数。 参考以下例子:

22 / 7

你会惊讶这个运算出的值是3。 这是因为如果你只使用整数表达式,Swift运算出结果也会是一个整数。 在这种情况下,结果是四舍五入到下一个整数。

你可以在Swift中设置使用小数表达式运算，操作如下:

22.0 / 7.0

这一次,结果是3.142857142857143。

余数运算

这四个操作到目前为止你看过的很简单易懂的运算,因为他们伴随你生活的大部分时间。 Swift也可以使用更复杂的运算,都是标准的数学运算,只是不太常见的。 现在让我们来了解他们。

第一个是余数运算 也称为模运算。 在这部分,分母模分子的次数,加上一个余数。 这正是余数运算的结果。 例如,10模3 = 1,因为三次3除以10,余数为1。 在Swift,余数运算符号为% ,操作如下: 

28 % 10

在这种情况下,结果= 8,因为10模28两次剩下的8。

命名数据

简单说，计算机编程就是在操作数据。 记住,你在屏幕中所看到的一切都能被压缩成数字发给CPU。 有时你自己处理这些各种类型数字形式出现的数据,但其他时候数据有更复杂的形式,如文本、图像和集合

在你的swift代码,你可以给数据命名方便你引用。 使用一个关联的名称这表示这个名字是指什么样 类型 的数据,如文本、数字或日期.

常量

看看这个:

let number: Int = 10

这个表明一个常数number 是 Int类型. 然后设置数量恒定的值 10.

注意:回想运算符号,这是另一个。 等号 =被称为 赋值运算符。.

类型 Int可以存储整数。与保存小数的方式是一样的:

let pi: Double = 3.14159

这是相似的 Int 常数,除了名字和类型是不同的。 这一次,常数是一个Double一个可以精确存储小数类型,。

还有一个类型也被叫做Float,储存小数精度较低于 Double. 事实上, Double 比 FloatFloat精确度高于两倍, 这就是为什么它 Double被放在第一位. Float 比 Double占用更少的内存但是一般来说,对数字的内存使用不是一个大问题,你会看到大多数地区使用在 Double

一旦你确定一个常数,你不能改变它的数据。 例如,请看下面的代码:

number = 0

这段代码会产生一个错误:

Cannot assign to value: 'number' is a 'let' constant

在Xcode中,你会看到错误这样显示出来:

常数对于值来说非常有用，因为它不会改变。 举个例子,如果你是模型飞机,需要随时知道的可用的席位总数,您可以使用一个常数。

你甚至可以使用一个常数来计算一个人的年龄。 尽管他们的生日到来时，年龄将会改变,你可能只会在这种特殊情况下关心他们的年龄。

变量

通常你想改变一个名字背后的数据。 举个例子,如果你是跟踪你的银行存款和提款账户余额,你要使用一个变量,而不是一个常数

如果您的程序的数据从未改变,那么这将是一个相当无聊的程序! 但如您所见,常数的数据是不可能改变的。

当你知道你需要改变一些数据,你应该使用一个变量来表示数据,而不是一个常数。 你确立一个变量,方法如下:

var variableNumber: Int = 42

只有第一部分的表述是不同的:你设定常量使用 let,而您使用的var作为变量.

一旦你设定一个变量,你可以随意改变,只要是相同的类型。 例如,改变上述变量设定,你可以这样做:

var variableNumber: Int = 42
variableNumber = 0
variableNumber = 1_000_000

改变一个变量,您只需指定一个新值。

使用有意义的名字

总是尽量为变量和常量选择有意义的名称。 好名字可以作为文档使代码容易识别。

一个好名字会具体地说 描述了变量或常数代表什么。 这里有一些好名字的例子:

  • personAge
  • numberOfPeople
  • gradePointAverage

坏名字经常只是因为描述性不够。 这里有一些例子不好的名字:

  • a
  • temp
  • average

关键是要确保你再使用变量或常数是会了解它们是关于什么的。 不要错误的认为你有一个可靠的记忆! 计算机编程中一般一两天之后回顾自己的代码，但很多时候却忘记了它。 为了使自己更容易记住，给你的变量和常量一个直观,准确的名称吧。

同时,注明上面的名字是如何取得。 在Swift中, 驼峰式大小写 的名字是很常见的。正常情况下你变量和常量,的名字遵循如下原则:

  • 从一个小写字母开始。
  • 如果名称是由多个单词,以每一个词一个大写字母开始组合它们。
  • 如果其中的一个单词是一个缩写,一样写出整个缩写(例如:sourceURL 和 urlDescription)

在Swift中,您甚至可以使用完整的Unicode字符。 例如,您可以设定一个变量如下:

var : Int = -1

你可能会笑,但谨慎使用这样的特殊字符。 它们更难输入,因此可能会让你比痛苦多于欢乐。

像这样的特殊字符可能储存的数据更有意义你而不是Swift代码。

递增和递减

一个常见的操作,您将需要是能够盈亏变量。 在Swift中,操作如下所示:

var counter: Int = 0

counter += 1
// counter = 1

counter -= 1
// counter = 0

counter变量开始是0. 增加设置它的值为1,然后衰减设置它的值为 0.

这些运算符类似于赋值运算符(=),但是他们还执行加法或减法。 他们把变量的当前值,添加或减去给定值,并将结果分配给变量。

换句话说,上面的代码是缩写如下:

var counter: Int = 0
counter = counter + 1
counter = counter - 1

类似地, *= 和 /=运营商做乘法和除法的等效,分别为:

var counter: Int = 10

counter *= 3  // same as counter = counter * 3
// counter = 30

counter /= 2  // same as counter = counter / 2
// counter = 15
